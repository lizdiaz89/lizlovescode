Responsive Web Design

freecodecamp

I highly recomend checking this website out. You can learn many programing lanuages. (e.g. Javascript, HTML, CSS, PYTHON,...and more.) Also freecodecamp lets you code along while learn. If you don't understand a lesson or have trouble completing the task on a lesson you have the ability to watch a tutorial of the lesson, which I find pretty cool and extremely helpful. This website is worth checking out and been very helpful during the beginning stages of my coding journey. They do have a youtube so I extremely recomend subscribing to their channel.

Coding for Beginners

Coding for beginners 1 Coding for beginners 2

I took these two seperated courses on udemy, and it is a paid course, but honestly it was worth buying. These courses give in depth lectures about the fundementals of coding and tutorials on HTML, CSS, & PYTHON. I paid for this course because I trying to brush up on my HTML skills and I also want a better "stuctured" understanding on what path I should take on my web development journey. I recomend checking this course out, the courses on Udemy are cheap and they offer deals for new subscribers.

Fullstack Web Development
